PC (Program Counter): 기억장치로 부터 읽혀질 다음 명령어의 주소를 저장
MAR (Memory Address Reg) : MBR의 주소를 저장
MBR (Memory Buffer Reg) : 기억장치에 저장될 단어를 가지고 있거나 기억 장치로 부터 읽혀진 다음 단어를 저장
IR (Intruction Reg) : 실행될 명령어의 연산코드를 저장

동작순서 : CPU는 기억장치(RAM)와 메모리를 교환한다.
1. PC가 첫 번째 명령어의 주소 x를 가지고 있다.
2. PC에 있는 주소가 MAR로 적재
3. 이때 MAR에 있는 주소가 main memory 주소를 가리킨다.
4. 그 주소에 해당하는 값을 MBR에 저장
5. 해당하는 값을 IT에 전달. 이 때 PC가 한 개 증가(다음 주소를 읽어야 하기 때문)
6. 제어장치가 명령어를 해독 후, AC에 data를 저장하거나 연산.
